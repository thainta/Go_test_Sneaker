# Setup

Step-1: Creating & activating venv
Windows:

```cmd
  py -3 -m venv .venv
  .venv\Scripts\activate
```

Step-2: Installing Dependencies

```bash
  pip install -r requirements.txt
```

Step-3: Running application
Windows:

```bash
  > flask run
```
